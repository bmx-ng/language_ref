===========
DisableMenu
===========

DisableMenu - 

Description
===========

.. code-block:: blitzmax

    DisableMenu( menu:TGadget )

Disable a menu so it cannot be selected.
#UpdateWindowMenu should be called where appropriate after changing a menu's status for the changes
to become visible.

Parameters
==========

Return Values
=============

Nothing.
