=======
Desktop
=======

Desktop - 

Description
===========

.. code-block:: blitzmax

    Desktop:TGadget()

Return a gadget representing the system's desktop.
This is particularly useful for finding the dimensions of the desktop using #GadgetWidth / #ClientWidth or #GadgetHeight / #ClientHeight.

Parameters
==========

Return Values
