===========
GadgetPaste
===========

GadgetPaste - 

Description
===========

.. code-block:: blitzmax

    GadgetPaste( gadget:TGadget )

Perform a paste operation on a gadget.
This is most commonly used on @TextAreas to paste text into the gadget from the clipboard.
