SetGadgetTextColor - 

Description
===========

.. code-block:: blitzmax

    SetGadgetTextColor( gadget:TGadget,r,g,b )

Set a gadget's foreground color.
The @{r}ed, @{g}reen and @{b}lue components should be in the range 0 to 255.
