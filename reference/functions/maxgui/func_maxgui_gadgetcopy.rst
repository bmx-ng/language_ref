==========
GadgetCopy
==========

GadgetCopy - 

Description
===========

.. code-block:: blitzmax

    GadgetCopy( gadget:TGadget )

Perform a copy operation on a gadget.
This is most commonly used on @TextAreas to copy text that is currently selected.
