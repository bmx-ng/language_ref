============
ActiveGadget
============

ActiveGadget - 

Description
===========

.. code-block:: blitzmax

    ActiveGadget:TGadget()

Return the currently active gadget.
See Also: #ActivateGadget.

Parameters
==========

Return Values
=============

The gadget that currently has the keyboard focus. Returns #Null if no MaxGUI gadget has focus.
