================
SetMaxWindowSize
================

SetMaxWindowSize - 

Description
===========

.. code-block:: blitzmax

    SetMaxWindowSize( window:TGadget,w,h )

Set a window's maximum size.
Particularly useful for resizable windows (those created with the WINDOW_RESIZABLE flag, see #CreateWindow).
