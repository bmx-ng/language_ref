SetMinWindowSize

SetMinWindowSize - 

Description
===========

.. code-block:: blitzmax

    SetMinWindowSize( window:TGadget,w,h )

Set a window's minimum size.
Particularly useful for resizable windows (those created with the WINDOW_RESIZABLE flag, see #CreateWindow).
