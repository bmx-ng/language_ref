=====
Delay
=====

Delay - 

Description
===========

.. code-block:: blitzmax

    Delay( millis )

Wait for a given number of milliseconds

#Delay suspends program execution for at least @millis milliseconds.<br>
<br>
A millisecond is one thousandth of a second.

Parameters
==========

Return Values
=============

Nothing.
