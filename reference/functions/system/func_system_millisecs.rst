=========
MilliSecs
=========

MilliSecs - 

Description
===========

.. code-block:: blitzmax

    MilliSecs()

Get millisecond counter

#MilliSecs returns the number of milliseconds elapsed since the computer
was turned on.<br>
<br>
A millisecond is one thousandth of a second.

Parameters
==========

Return Values
=============

Milliseconds since computer turned on.
