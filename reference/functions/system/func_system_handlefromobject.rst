HandleFromObject
================

HandleFromObject - 

Description
===========

.. code-block:: blitzmax

    HandleFromObject( obj:Object )

Convert object to integer handle

After converting an object to an integer handle, you must later
release it using the #Release command.

Parameters
==========

Return Values
=============

An integer object handle
