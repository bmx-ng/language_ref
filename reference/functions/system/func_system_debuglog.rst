========
DebugLog
========

DebugLog - 

Description
===========

.. code-block:: blitzmax

    DebugLog( message$ )

Write a string to debug log
If there is no debugger present, this command is ignored.

Parameters
==========

Return Values
=============

Nothing.

Examples
========

.. code-block:: blitzmax

    ' launchdir.bmx
    
    Print "This program was launched from "+LaunchDir$
