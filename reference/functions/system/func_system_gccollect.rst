=========
GCCollect
=========

GCCollect - 

Description
===========

.. code-block:: blitzmax

    GCCollect()

Run garbage collector

This function will have no effect if the garbage collector has been
suspended due to #GCSuspend.

Parameters
==========

Return Values
=============

The amount of memory, in bytes, collected.
