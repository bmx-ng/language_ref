===========
CurrentTime
===========

CurrentTime - 

Description
===========

.. code-block:: blitzmax

    CurrentTime$()

Get current time

Returns the current time in the format: HH:MM:SS (i.e. 14:31:57).

Parameters
==========

Return Values
=============

The current time as a string

Examples
========

.. code-block:: blitzmax

    ' currenttime.bmx
    
    Print "The time is "+CurrentTime$()
