===========
CurrentDate
===========

CurrentDate - 

Description
===========

.. code-block:: blitzmax

    CurrentDate$()

Get current date

Returns the current date in the format: DD MON YYYY (i.e. 10 DEC 2000).

Parameters
==========

Return Values
=============

The current date as a string

Examples
========

.. code-block:: blitzmax

    ' currentdate.bmx
    
    Print "The date is "+CurrentDate$()
