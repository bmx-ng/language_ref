LoadPixmapJPeg
==============

LoadPixmapJPeg - 

Description
===========

.. code-block:: blitzmax

    LoadPixmapJPeg:TPixmap( url:Object )

Load a Pixmap in JPeg format

#LoadPixmapJPeg loads a pixmap from @url in JPeg format.

If the pixmap cannot be loaded, Null is returned.
