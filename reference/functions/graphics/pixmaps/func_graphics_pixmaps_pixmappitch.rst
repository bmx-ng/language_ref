===========
PixmapPitch
===========

PixmapPitch - 

Description
===========

.. code-block:: blitzmax

    PixmapPitch( pixmap:TPixmap )

Get pixmap pitch

Pitch refers to the difference, in bytes, between the start of one row of pixels and the start of the next row.

Parameters
==========

Return Values
=============

The pitch, in bytes, of @pixmap
