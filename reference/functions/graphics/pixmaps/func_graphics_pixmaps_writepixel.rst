==========
WritePixel
==========

WritePixel - 

Description
===========

.. code-block:: blitzmax

    WritePixel( pixmap:TPixmap,x,y,argb )

Write a pixel to a pixmap

The 32 bit @argb value contains the following components:

[ bits 24-31 | pixel alpha
* bits 16-23 | pixel red
* bits 8-15 | pixel green
* bits 0-7 | pixel blue
]

==========
