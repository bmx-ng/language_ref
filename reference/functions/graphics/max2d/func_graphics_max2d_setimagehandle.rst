==============
SetImageHandle
==============

SetImageHandle - 

Description
===========

.. code-block:: blitzmax

    SetImageHandle( image:TImage,x#,y# )

Set an image's handle to an arbitrary point

An image's handle is subtracted from the coordinates of #DrawImage before
rotation and scale are applied.
