========
SetScale
========

SetScale - 

Description
===========

.. code-block:: blitzmax

    SetScale( scale_x#,scale_y# )

Set current scale

@scale_x and @scale_y multiply the width and height of drawing
commands where 0.5 will half the size of the drawing and 2.0 is equivalent
to doubling the size.

Parameters
==========

Return Values
=============

Nothing.
