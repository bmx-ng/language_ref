===========
SetClsColor
===========

SetClsColor - 

Description
===========

.. code-block:: blitzmax

    SetClsColor( red,green,blue )

Set current #Cls color

The @red, @green and @blue parameters should be in the range of 0 to 255.

The default cls color is black.

Parameters
==========

Return Values
=============

Nothing.
