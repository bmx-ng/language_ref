==========
TextHeight
==========

TextHeight - 

Description
===========

.. code-block:: blitzmax

    TextHeight( text$ )

Get height of text

This command is useful for calculating vertical alignment of text when using
the #DrawText command.

Parameters
==========

Return Values
=============

the height, in pixels, of @text based on the current image font.
