========
SetColor
========

SetColor - 

Description
===========

.. code-block:: blitzmax

    SetColor( red,green,blue )

Set current color

The #SetColor command affects the color of #Plot, #DrawRect, #DrawLine, #DrawText,
#DrawImage and #DrawPoly.

The @red, @green and @blue parameters should be in the range of 0 to 255.

Parameters
==========

Return Values
=============

Nothing.
