============
SetTransform
============

SetTransform - 

Description
===========

.. code-block:: blitzmax

    SetTransform( rotation#=0,scale_x#=1,scale_y#=1 )

Set current rotation and scale

SetTransform is a shortcut for setting both the rotation and
scale parameters in Max2D with a single function call.

Parameters
==========

Return Values
