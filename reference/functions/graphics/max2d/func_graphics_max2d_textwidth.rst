=========
TextWidth
=========

TextWidth - 

Description
===========

.. code-block:: blitzmax

    TextWidth( text$ )

Get width of text

This command is useful for calculating horizontal alignment of text when using
the #DrawText command.

Parameters
==========

Return Values
=============

the width, in pixels, of @text based on the current image font.
