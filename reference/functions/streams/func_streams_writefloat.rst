WriteFloat - 

Description
===========

.. code-block:: blitzmax

    WriteFloat( stream:TStream,n# )

Write a Float to a stream
#WriteFloat writes 4 bytes to @stream.
A TStreamWriteException is thrown if not all bytes could be written
