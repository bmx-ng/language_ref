===========
WriteString
===========

WriteString - 

Description
===========

.. code-block:: blitzmax

    WriteString( stream:TStream,str$ )

Write a String to a stream

Each character in @str is written to @stream.

A #TStreamWriteException is thrown if not all bytes could be written.

Parameters
==========

Return Values
=============
