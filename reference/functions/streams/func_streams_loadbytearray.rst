LoadByteArray
=============

LoadByteArray - 

Description
===========

.. code-block:: blitzmax

    LoadByteArray:Byte[]( url:Object )

Load a Byte array from a stream

The specified @url is opened for reading, and each byte in the resultant stream
(until eof of reached) is read into a byte array.

Parameters
==========

Return Values
=============

A Byte array
