CreateBankStream
================

CreateBankStream - 

Description
===========

.. code-block:: blitzmax

    CreateBankStream:TBankStream( bank:TBank )

Create a bank stream

A bank stream allows you to read data into or out of a bank. A bank stream extends a stream so
can be used in place of a stream.
