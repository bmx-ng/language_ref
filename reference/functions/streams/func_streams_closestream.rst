===========
CloseStream
===========

CloseStream - 

Description
===========

.. code-block:: blitzmax

    CloseStream( stream:TStream )

Close a stream

All streams should be closed when they are no longer required.
Closing a stream also flushes the stream before it closes.

Parameters
==========

Return Values
