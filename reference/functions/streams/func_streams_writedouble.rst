WriteDouble - 

Description
===========

.. code-block:: blitzmax

    WriteDouble( stream:TStream,n! )

Write a Double to a stream
#WriteDouble writes 8 bytes to @stream.
A TStreamWriteException is thrown if not all bytes could be written
