ReadByte
========

ReadByte - 

Description
===========

.. code-block:: blitzmax

    ReadByte( stream:TStream )

Read a Byte from a stream
#ReadByte reads a single Byte from @stream.
A TStreamReadException is thrown If there is not enough data available.

Parameters
==========

Return Values
