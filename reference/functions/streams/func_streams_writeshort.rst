WriteShort

WriteShort - 

Description
===========

.. code-block:: blitzmax

    WriteShort( stream:TStream,n )

Write a Short to a stream
#WriteShort writes 2 bytes to @stream.
A TStreamWriteException is thrown if not all bytes could be written
