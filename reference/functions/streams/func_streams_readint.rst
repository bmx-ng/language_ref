ReadInt
=======

ReadInt - 

Description
===========

.. code-block:: blitzmax

    ReadInt( stream:TStream )

Read an Int from a stream
#ReadInt reads 4 bytes from @stream.
A TStreamReadException is thrown If there is not enough data available.

Parameters
==========

Return Values
