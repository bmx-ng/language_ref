========
ReadLong
========

ReadLong - 

Description
===========

.. code-block:: blitzmax

    ReadLong:Long( stream:TStream )

Read a Long from a stream
#ReadLong reads 8 bytes from @stream.
A TStreamReadException is thrown If there is not enough data available.

Parameters
==========

Return Values
=============

A Long value
