ReadFloat

ReadFloat - 

Description
===========

.. code-block:: blitzmax

    ReadFloat#( stream:TStream )

Read a Float from a stream
#ReadFloat reads 4 bytes from @stream.
A TStreamReadException is thrown If there is not enough data available.

==========
