========
WriteInt
========

WriteInt - 

Description
===========

.. code-block:: blitzmax

    WriteInt( stream:TStream,n )

Write an Int to a stream
#WriteInt writes 4 bytes to @stream.
A TStreamWriteException is thrown if not all bytes could be written

==========
