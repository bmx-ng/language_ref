========
JoyWheel
========

JoyWheel - 

Description
===========

.. code-block:: blitzmax

    JoyWheel#( port=0 )

Reports the position of the joystick's YAW axis if supported.

Parameters
==========

Return Values
=============

Zero if the joystick is centered, -1.0 if Left, 1.0 if Right or a value inbetween.
