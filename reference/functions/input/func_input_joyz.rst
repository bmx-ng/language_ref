====
JoyZ
====

JoyZ - 

Description
===========

.. code-block:: blitzmax

    JoyZ#( port=0 )

Reports the position of the joystick's Z axis if supported.

Parameters
==========

Return Values
=============

Zero if the joystick is centered, -1.0 if Up, 1.0 if Down or a value inbetween.
