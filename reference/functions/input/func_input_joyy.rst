====
JoyY
====

JoyY - 

Description
===========

.. code-block:: blitzmax

    JoyY#( port=0 )

Reports the vertical position of the joystick.

Parameters
==========

Return Values
=============

Zero if the joystick is centered, -1.0 if Up, 1.0 if Down or a value inbetween.
