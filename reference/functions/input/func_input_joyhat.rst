======
JoyHat
======

JoyHat - 

Description
===========

.. code-block:: blitzmax

    JoyHat#( port=0 )

Reports the position of the joystick's HAT controller if supported.

Parameters
==========

Return Values
=============

-1.0 if the joystick is centered, and values between 0.0, 0.25, 0.5 and 0.75 for the directions Up, Right, Down, Left respectively.
