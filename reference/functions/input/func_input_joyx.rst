====
JoyX
====

JoyX - 

Description
===========

.. code-block:: blitzmax

    JoyX#( port=0 )

Reports the horizontal position of the joystick.

Parameters
==========

Return Values
=============

Zero if the joystick is centered, -1 if Left, 1 if Right or a value inbetween.
