=========
FlushKeys
=========

FlushKeys - 

Description
===========

.. code-block:: blitzmax

    FlushKeys()

Flush key states and character queue.

#FlushKeys resets the state of all keys to 'off', and resets the character queue
used by #GetChar.

Parameters
==========

Return Values
