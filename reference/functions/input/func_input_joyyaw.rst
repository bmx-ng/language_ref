======
JoyYaw
======

JoyYaw - 

Description
===========

.. code-block:: blitzmax

    JoyYaw#( port=0 )

Reports the position of the joystick's YAW axis if supported.

Parameters
==========

Return Values
=============

Zero if the joystick is centered, -1.0 if Up, 1.0 if Down or a value inbetween.
