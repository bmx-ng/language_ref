====
JoyR
====

JoyR - 

Description
===========

.. code-block:: blitzmax

    JoyR#( port=0 )

Reports the position of the joystick's R axis if supported.

Parameters
==========

Return Values
=============

Zero if the joystick is centered, -1.0 if Up, 1.0 if Down or a value inbetween.
