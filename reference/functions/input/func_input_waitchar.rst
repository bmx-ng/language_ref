========
WaitChar
========

WaitChar - 

Description
===========

.. code-block:: blitzmax

    WaitChar()

Wait for a key press

#WaitChar suspends program execution until a character is available from #GetChar. This
character is then returned to the application.

Parameters
==========

Return Values
=============

The character code of the pressed key
