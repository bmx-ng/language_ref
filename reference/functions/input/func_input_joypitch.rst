========
JoyPitch
========

JoyPitch - 

Description
===========

.. code-block:: blitzmax

    JoyPitch#( port=0 )

Reports the position of the joystick's PITCH axis if supported.

Parameters
==========

Return Values
=============

Zero if the joystick is centered, -1.0 if Up, 1.0 if Down or a value inbetween.
