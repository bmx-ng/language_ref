.. _func_file_filesize:

========
FileSize
========

FileSize - 

Description
===========

.. code-block:: blitzmax

    FileSize( path$ )

Get file size

Parameters
==========

Return Values
=============

Size, in bytes, of the file at @path, or -1 if the file does not exist

Examples
========

.. code-block:: blitzmax

    ' filesize.bmx
    
    ' the following prints the size of this source file in bytes
    
    print filesize("filesize.bmx")
