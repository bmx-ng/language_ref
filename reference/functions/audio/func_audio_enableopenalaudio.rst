=================
EnableOpenALAudio
=================

EnableOpenALAudio - 

Description
===========

.. code-block:: blitzmax

    EnableOpenALAudio()

Enable OpenAL Audio

After successfully executing this command, OpenAL audio drivers will be added
to the array of drivers returned by #AudioDrivers.

Parameters
==========

Return Values
=============

True if successful
