========
PeekByte
========

PeekByte - 

Description
===========

.. code-block:: blitzmax

    PeekByte( bank:TBank,offset )

Peek a byte from a bank

A byte is an unsigned 8 bit value with a range of 0..255.

Parameters
==========

Return Values
=============

The byte value at the specified byte offset within the bank
