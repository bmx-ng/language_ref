UnlockBank
==========

UnlockBank - 

Description
===========

.. code-block:: blitzmax

    UnlockBank( bank:TBank )

Unlock a bank's memory block

After you have finished with a bank's memory block, you must use #UnlockBank
to return it to the bank.
