========
PeekLong
========

PeekLong - 

Description
===========

.. code-block:: blitzmax

    PeekLong:Long( bank:TBank,offset )

Peek a long integer from a bank

A long is a 64 bit integer that requires 8 bytes of memory.

Parameters
==========

Return Values
=============

The long integer value at the specified byte offset within the bank
