========
SaveBank
========

SaveBank - 

Description
===========

.. code-block:: blitzmax

    SaveBank( bank:TBank,url:Object )

Save a bank

#SaveBank writes it's entire contents to a @url. If the @url is a file path a new
file is created.

Parameters
==========

Return Values
=============

True if successful.
