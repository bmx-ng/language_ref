=========
PeekShort
=========

PeekShort - 

Description
===========

.. code-block:: blitzmax

    PeekShort( bank:TBank,offset )

Peek a short from a bank

A short is an unsigned 16 bit (2 bytes) value with a range of 0..65535.

Parameters
==========

Return Values
=============

The short value at the specified byte offset within the bank
