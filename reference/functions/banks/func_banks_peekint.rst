PeekInt
=======

PeekInt - 

Description
===========

.. code-block:: blitzmax

    PeekInt( bank:TBank,offset )

Peek an int from a bank

An int is a signed 32 bit value (4 bytes).

Parameters
==========

Return Values
=============

The int value at the specified byte offset within the bank
